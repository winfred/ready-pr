# Ready label

Listens to github events from your PRs and will mark the Aha! Feature as ready/in development based on the label.


# Using

Clone this repo.

Basically, replace `winfred` with `yourgithubhandle`

Change the package.json file to have 
* an identifier unique to you
* a name unique to you


Change src/prListener.js to check
* your github username instead of `winfred`
